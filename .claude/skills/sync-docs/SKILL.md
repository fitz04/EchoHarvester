# 문서 동기화

코드베이스의 현재 상태를 기준으로 모든 프로젝트 문서를 일괄 동기화한다.

## 절차

1. `echoharvester/` 아래 모든 Python 모듈을 스캔하여 현재 구현 상태 파악
2. `agent.md`를 읽어 기록된 진행 상태와 실제 코드를 비교
3. 불일치 항목을 TodoWrite 체크리스트로 정리
4. 각 문서를 순차적으로 업데이트:
   - `agent.md`: 작업 로드맵 체크박스, 현재 진행 상태, 이슈 테이블
   - 기타 기획/문서 파일이 있으면 함께 업데이트
5. 모든 문서 업데이트 완료 후 변경 요약 출력

## 규칙

- 부분 업데이트 금지 — 관련 문서 전체를 한 번에 완료
- 코드에서 확인된 사실만 기록 (추측 금지)
- 날짜는 실제 작업일 기준으로 기록
- 기존 형식과 스타일을 유지

## 출력 형식

```
## 문서 동기화 결과

### 변경된 파일
- (파일명): (변경 요약)

### 주요 업데이트
- (핵심 변경사항 나열)

### 동기화 완료 확인
- [ ] agent.md 업데이트 완료
- [ ] 모든 체크박스 상태 정확
- [ ] 이슈 테이블 최신화
```
