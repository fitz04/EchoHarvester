# 프로젝트 상태 확인

프로젝트의 현재 진행 상태를 요약 리포트로 출력한다.

## 절차

1. `agent.md`를 읽어 현재 Phase와 작업 로드맵 파악
2. `git log --oneline -10`으로 최근 커밋 확인
3. `git status`로 미커밋 변경사항 확인
4. 코드 디렉토리(`echoharvester/`)와 `agent.md`의 진행 상태를 비교하여 불일치 확인

## 출력 형식

```
## 프로젝트 상태 요약

**현재 Phase**: Phase X - (제목)
**상태**: (진행중/완료)

### 완료된 항목
- (체크된 항목 나열)

### 진행 중
- (현재 작업 중인 항목)

### 다음 단계
- (다음으로 해야 할 항목 1-3개)

### 최근 변경 (커밋)
- (최근 커밋 5개)

### 미커밋 변경사항
- (있으면 나열, 없으면 "없음")

### 문서-코드 불일치
- (불일치 항목이 있으면 나열, 없으면 "없음")
```
